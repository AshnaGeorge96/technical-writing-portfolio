# Vehicle Tracking Application – User Guide

## Overview
The Vehicle Tracking Application allows users to track their vehicles in real time,
view live location updates, and receive important notifications related to vehicle movement.

This guide explains how to use the application features step by step.

---

## System Requirements
- Android 8.0 or later / iOS 13 or later
- Active internet connection
- Valid user account credentials

---

## Getting Started

### Create an Account
1. Open the Vehicle Tracking Application.
2. Tap on **Sign Up**.
3. Enter your registered mobile number or email address.
4. Verify using the OTP sent to your device.
5. Create a secure password and complete registration.

### Login
1. Open the app.
2. Enter your registered email or mobile number.
3. Enter your password.
4. Tap **Login** to access the dashboard.

---

## Dashboard Overview
After logging in, the dashboard displays:
- Current vehicle status
- Live location on the map
- Last updated time
- Quick access to tracking features

---

## Real-Time Vehicle Tracking
1. Select a vehicle from the dashboard.
2. The live map will display the vehicle’s current location.
3. Location updates refresh automatically at regular intervals.
4. Zoom and pan the map to view surrounding areas.

---

## Notifications
The application sends notifications for:
- Vehicle movement
- Ignition status changes
- Geo-fence alerts (if enabled)

To manage notifications:
1. Go to **Settings**.
2. Enable or disable notification preferences.

---

## Common Issues & Troubleshooting

### Location Not Updating
- Ensure mobile data or Wi-Fi is enabled.
- Check GPS permissions for the app.
- Restart the application if the issue continues.

### Unable to Login
- Verify login credentials.
- Use **Forgot Password** to reset your password.
- Check internet connectivity.

---

## Security & Privacy
- User data is securely stored.
- Location information is accessible only to authorized users.
- Always log out when using shared devices.
